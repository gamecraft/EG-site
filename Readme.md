
# HTML5 Express Boilerplate ("H5EB")

Get started with a modern boilerplate for [Express](http://expressjs.com)!

## Requirements
  * [node](https://github.com/joyent/node)
  * [npm](https://github.com/isaacs/npm)

# Quick Start

      npm install -g h5eb
      h5eb project-name && cd project-name
      npm install -d
      node server.js
